# Creating Structure Label Styles

The collection of all structure label styles in a document is found in the `CivilDocument.Styles.LabelStyles.StructureLabelStyles` property, which is a `LabelStylesStructureRoot`  object. .

Note:

The label style of a particular structure cannot be set using the .NET API.
