# Data Shortcuts

This chapter covers Civil 3D data references, and how to manage them using the Autodesk Civil 3D .NET API.

There are currently two kinds of operations possible with the data shortcut API. You can:

* Manage working folders and project folders
* Manage the status of data referenced elements in a drawing.

Tip: Data Shortcut APIs are available to Autodesk Construction Cloud workflows. This functionality requires access to
[Autodesk Collaboration for Civil 3D](https://beehive.autodesk.com/community/service/rest/cloudhelp/resource/cloudhelpchannel/guidcrossbook/?v=2026&p=CIV3D&l=ENU&accessmode=live&guid=GUID-6C3CD2CD-4ABC-4A33-A9FF-5EBD6E67A8CC).
