# Run the Plugin in Civil 3D

1. Start Autodesk Civil 3D, and type
   `NETLOAD` at the command line. In the Choose .NET Assembly dialog, browse to your assembly DLL.
2. Open or create a document in Autodesk Civil 3D that contains alignments and sites.
3. Type
   `HelloCivil3D` at the command line, and you will see the command output:
