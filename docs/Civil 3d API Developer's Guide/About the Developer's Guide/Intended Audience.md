# Intended Audience

The
Autodesk Civil 3D API Developer’s Guide is designed for developers who want to customize Autodesk® Civil 3D® or create applications using the underlying APIs. It can also be used for creating macros to automate repetitive tasks for
Autodesk Civil 3D users and for developers of custom subassemblies.
