# Sample Program

## SurfacePointsSample.dvb

**<installation-directory>\Sample\Civil 3D API\Vba\SurfacePoints\SurfacePointsSample.dvb**

The sample code from this section can be found in context in the SurfacePointsSample.dvb program. The Points module contains functions that load points from a text file, create points manually, use description keys, deal with point groups, and use points to modify a surface. The PointStyles module demonstrates the creation of a point style and a point label style.
