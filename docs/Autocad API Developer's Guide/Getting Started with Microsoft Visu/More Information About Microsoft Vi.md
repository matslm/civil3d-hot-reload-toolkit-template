# More Information About Microsoft Visual Studio (.NET)

More information on the Microsoft Visual Studio development environment and .NET programming languages, see the help files provided by Microsoft. To access the help files, choose one of the options from the Help menu in Microsoft Visual Studio. You might also consider obtaining a book published by Microsoft ® Press or a third-party publisher on using C#.

Note: While VB.NET is a supported programming language, it is recommended to learn and use C# as it is more prevalent in the industry.
