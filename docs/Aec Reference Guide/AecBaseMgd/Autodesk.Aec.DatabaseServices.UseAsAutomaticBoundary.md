# Autodesk.Aec.DatabaseServices.UseAsAutomaticBoundary

## Summary
Defines identifiers that indicate whether the object can act as an automatic boundary.

## Fields

### NotApplicableToThisObject
Automatic Boundries do not apply to this object type.

### Yes
This object is set to act as an automatic boundary.

### No
This object is not set to act as an automatic boundary.

### ByStyle
This object relies on the value in it's style to determine whether it is set to act as an automatic boundary.
